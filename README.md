# 검출 영역을 어떻게 읽을 것인가  
<img src='./영역을_어떻게_읽을것인가.png' />  
▶ 해당 검출 영역의 모든 픽셀값이 70 이하  

▶ 해당 검출 영역의 픽셀값 평균이 70 이하  

▶ 픽셀의 평균값이 제일 작은 영역  

▶ 픽셀의 분산값이 큰 영역  

▶ 픽셀값을 오름차순 정렬하여 상위 50개 평균값의 최소값

▶   영역에 따라 가중치를 부여
